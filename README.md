Overview:
the grid analyzer is a java program thats designed to process and analyze 2d arrays.
it allows user to define the grid dimensions.
grid has a minimum 6x6 size.
has a menu driven interface thats user friendly.
its against user errors incase of user misuse.

Implemented Algoriths:
grid display- prints out grid
row and col sum- prints out sum of every row and every col
global max/min- finds max and min value in entire grid and their indexes 
frequency counter- counts occurences of target and values greater than threshold
row/col comparison- compare row with most sum and col with least sum
pattern detection- finds if any row is strictly increasing 
grid transformations- allows user to rotate a row, swap 2 rows, or reverse a column
subgrid processing- allows user to find sum of specific neighbourhood or area within the grid
boundary and diagonal analysis- finds boundary and main and secondary diagonals elements
data validation- checks if any duplicates are in grid

How to use:
1) initialize row and col size
2) enter data of each coordinate
3) select option from main menu
4) display grid
5) enter 0 to exit or close program
